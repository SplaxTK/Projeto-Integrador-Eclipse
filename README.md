##
O objetivo do PI é integrar as ODS da ONU em um site.
O nosso projeto utiliza a ODS 7 para criar um site que ajude na escolha de energias renováveis no dia a dia das pessoas,
fazendo um mundo mais saudável, limpo e sustentável. 
#

#
Para isso, nossos integrantes dividiram várias tarefas na criação e conceituação do site.
O site já tem várias funções prontas, como as informações de cada energia, a barra de pesquisa,
as opções de acessibilidade e a de entrar em contato com diferentes empresas de energia em variadas cidades.
